# 📝 My Blog Website

A modern, minimal blog website built using HTML, CSS, and JavaScript.  
Users can write, read, search, and manage articles with a clean UI and smooth experience.

---

## Live Preview: ( https://kfizza1.github.io/Blog-Site/)

---

## 🚀 Features

- ✍️ Write and publish articles
- 📰 Featured article section (latest post highlighted)
- 🔍 Real-time search functionality
- 📚 Archive page for all articles
- 📖 Full article reading view
- 🗑️ Delete articles
- 💾 Data stored in localStorage (no backend needed)
- 📱 Responsive design

---


## ⚙️ How It Works

### 📌 Data Storage
- Articles are stored in localStorage
- Default articles (SEED data) load on first visit
- No database or backend required

---

### 🧠 Core Functionalities

#### 1. Create Article
- User fills the form
- Article is saved in localStorage
- Appears instantly on homepage

#### 2. Read Article
- Click any article
- Opens full view with details

#### 3. Search
- Filters articles by title, content, or excerpt

#### 4. Delete Article
- Removes article from storage
- UI updates instantly
